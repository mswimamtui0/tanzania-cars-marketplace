# 🚗 Tanzania Car Marketplace

A professional car marketplace platform built with Django, similar to AutoTrader UK, designed specifically for the Tanzanian market.

## 🌟 Features

### User Roles
- **Admin** - Full system control
- **Dealer** - List and manage cars
- **Yard Manager** - Manage physical car yards
- **Buyer** - Browse and purchase cars
- **Agent/Broker** - Assist buyers and sellers

### Core Features
- 🔐 Multi-role authentication system
- 🚗 Car listings with images
- 📱 WhatsApp integration for chat
- ✅ Seller verification levels (Level 1-3)
- 💰 Listing packages (Normal, Featured, Premium)
- 🔍 Advanced search and filters
- ❤️ Wishlist functionality
- 📊 Professional admin dashboard
- 💬 Chat routing based on trust level
- 🔧 Vehicle inspection requests
- 📈 Car valuation tool
- 🏢 Yard management system

### Payment System
- M-Pesa integration ready
- Multiple listing packages
- Agent commission tracking

## 🛠️ Technology Stack

- **Backend**: Django 4.2.7
- **Database**: SQLite (dev) / PostgreSQL (production)
- **Frontend**: Bootstrap 5, JavaScript, Chart.js
- **Payments**: M-Pesa, Stripe ready
- **Deployment**: Ready for Heroku, PythonAnywhere, or VPS

## 📋 Prerequisites

- Python 3.11 or 3.12 (Python 3.13 not recommended)
- pip
- Git
